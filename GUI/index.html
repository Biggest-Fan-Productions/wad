8-Track Live Looper

Controls Overview

Mute or unmute any track by pressing the number key for that track (1 - 8).

Record to a track by holding [record] and pressing the number key for the track you want to record to. Repeat this action to stop recording, or record to a different track. You can only record to one track at a time.

You can erase a track by holding [erase] and pressing the number key for the track you want to erase.

Toggle microphone input on or off by pressing [microphone].

Start or restart the looping animation by pressing [animate].

This application contains 4 different instruments which you can switch between at any time. The first three are synthesized with oscillators, while the fourth instrument is a drum kit, which includes several samples.
