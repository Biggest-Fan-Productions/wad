Configuration

This application is an 8-track live looper.

Controls Overview

Mute or unmute any track by pressing the number key for that track (1 - 8).

Record to a track by holding [record] while pressing the number key for that track. Repeat this action to stop recording, or record to a different track. You can only record to one track at a time.

Schedule a track to mute or unmute itself at the start of the next loop by holding [schedule] while pressing the number key for that track.

Schedule a track to record at the start of the next loop by holding [schedule] and [record] while pressing the number key for that track.
